Grocery Delivery Management System
Author: Susan Rogers

How to Run the Project in NetBeans

1. Open NetBeans.
2. Select File > Open Project.
3. Open the Grocery Delivery Management System project folder.
4. Build and run the project.
5. Log in using the sample accounts included in the DataStore class.

Software Used
- Java
- NetBeans IDE

Important Note
This version of the project uses in-memory data storage through Java ArrayLists.
No SQL or MySQL setup is required.
